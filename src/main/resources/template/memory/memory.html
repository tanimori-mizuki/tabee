千葉旅行
会社の同期3人で日帰り千葉ドライブに行ってきました。天気は晴れ！日差しも強くまさに夏！という1日でした。朝9時に川崎駅に集合し、そこからレンタカーに乗り千葉を目指しアクアラインへ。途中海ほたるに立ち寄り、一面に広がる大海原を眺めながら記念写真も。とにかく暑かったので早々に切り上げドライブ再開。
